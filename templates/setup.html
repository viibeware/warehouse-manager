Set a strong admin password

Replace the default admin password before anyone else gets in. This is the only required step — everything after it can be skipped and configured later.

  • At least 12 characters
  • Mix of lower / upper / number / symbol (any 3)
  • Not the word "admin"
  • Both fields match
Use STARTTLS
Standard for port 587. Disable for port 465 (implicit SSL).

All fields are optional. If you leave the host blank the wizard saves nothing and you can configure SMTP later in Settings → Admin → SMTP / Email.
