Refresh downloads the official chart for every origin prefix directly from ups.com, pacing requests a couple of seconds apart. Without “force” it only fetches charts that are missing or invalid. New charts go live the moment the run finishes. UPS typically updates rates in late December. If ups.com blocks the server (bot protection), charts can also be dropped into the data volume's zonechart/charts/ directory as NNN.xls files.
